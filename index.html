Descarca aplicatia (.exe)

De ce apare „Run anyway”?

Windows poate afisa mesajul „Windows protected your PC” atunci cand rulezi aplicatia pentru prima data.

Acest lucru este normal si apare deoarece aplicatia:

  • este creata de un dezvoltator independent
  • nu este semnata digital cu un certificat comercial
  • este descarcata de pe internet

Aplicatia NU contine virusi si este open-source. Codul poate fi verificat pe GitHub.

Pentru a rula aplicatia:

  1. Apasa More info
  2. Apasa Run anyway

Acest mesaj este standard pentru aplicatii de portofoliu si proiecte personale.
